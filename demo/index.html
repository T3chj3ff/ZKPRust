ZKPRust uses Schnorr Zero-Knowledge Proofs over Ristretto255 to verify identity cryptographically — no password ever leaves your browser.

🔒 Everything below runs in your browser. Zero network requests. Zero password transmission.
zkp-demo.js — powered by zkp_wasm.wasm (118 KB)
1

Register

Your password is hashed (SHA-256) in-browser. The public key — a Ristretto255 curve point — is derived. This is what your server stores.

2

Login

WASM generates a fresh 64-byte Schnorr proof. Only the proof is transmitted — never the password or its hash.

3

Verify

The server (or WASM here for demo) verifies the proof mathematically against the stored public key. No brute-force possible.
